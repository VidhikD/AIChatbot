
# AI Chatbot 💬

A simple AI chatbot built with Java, JavaFX, and the Hugging Face API.

## Features 🚀
- Dark mode & Light mode 🌗
- Animated UI ✨
- Uses Hugging Face API for AI responses 🤖
- JavaFX GUI
